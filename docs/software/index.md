# Software

The Glossarist ecosystem provides tools for every stage of terminology management.

| Project | Description |
|---------|-------------|
| [glossarist-ruby](/docs/software/glossarist-ruby) | Ruby gem implementing the Glossarist concept model |
| [glossarist-js](/docs/software/glossarist-js) | JavaScript SDK for GCR packages |
| [Concept Browser](/docs/software/concept-browser) | Interactive browser for terminology datasets |
| [Desktop App](/docs/software/desktop) | Native desktop viewer and editor for concept registries |
