# Desktop App

Desktop viewer and editor for concept registries. Manage concepts, propose changes, and review change requests from a native app on Windows, macOS, and Linux. See the [concept model docs](/docs/model/) for the data model, or the [desktop docs](/docs/desktop/) for detailed usage.

<ReleaseDownloader />

## Features

- View and browse concept registries with multi-language support
- Create, edit, and translate concept entries
- Propose changes through a structured change request workflow
- Review and accept/reject proposed changes as register manager
- Concept relationship graph view (early preview)
- Automatic synchronization with Git-based data repository
