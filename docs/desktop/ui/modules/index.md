# Modules

Module is selected at the top of main application window. Each of the modules allows to perform a specific task with the register or with the currently selected register item.

- [Module "List"](/docs/desktop/ui/modules/browse) — Shows (sub)registry contents as a list
- [Module "Create"](/docs/desktop/ui/modules/create) — Propose the authoritative language entry for an entirely new registry item
- [Module "Edit"](/docs/desktop/ui/modules/edit) — Propose changes to the authoritative language entry of currently selected registry item
- [Module "Translate"](/docs/desktop/ui/modules/translate) — Propose changes to the non-authoritative language entry of currently selected registry item
- [Module "Review"](/docs/desktop/ui/modules/review) — Review change request (for submitters, your own drafts; for registry managers, submitted proposals)
- [Module "Card"](/docs/desktop/ui/modules/view) — Shows a somewhat detailed view of currently selected item
- [Module "Graph"](/docs/desktop/ui/modules/map) — Shows registry items as a graph of nodes with edges signifying relationships (early preview)
