# Panels

Panels are located on the sides of the window.

- [Panel "Item"](/docs/desktop/ui/panels/item) — General registry item properties
- [Panel "Language"](/docs/desktop/ui/panels/language) — Choose language to view or edit
- [Panel "Relationships"](/docs/desktop/ui/panels/relationships) — View and manage relationships
- [Panel "Revisions"](/docs/desktop/ui/panels/revisions) — Lists revision history
- [Panel "Selected Revision"](/docs/desktop/ui/panels/selected-revision) — Describes the selected revision
- [Panel "My Draft Change Requests"](/docs/desktop/ui/panels/my-draft-change-requests) — Select which CR to add edits to
- [Panel "Concept Summary"](/docs/desktop/ui/panels/concept-summary) — Summary of selected concept revision
- [Sidebar](/docs/desktop/ui/panels/sidebar) — Collapsible sidebar
- [Top Panel](/docs/desktop/ui/panels/top-panel) — Collapsible top panel
