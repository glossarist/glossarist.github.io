# Interface Elements

What you see in the app and how it works.

::: info
The main window of Glossarist app shows a brief description of the element under your mouse cursor: find Help panel in bottom left corner.
:::

## Modules

- [Module "List"](/docs/desktop/ui/modules/browse) — Shows (sub)registry contents as a list
- [Module "Create"](/docs/desktop/ui/modules/create) — Propose the authoritative language entry for a new registry item
- [Module "Edit"](/docs/desktop/ui/modules/edit) — Propose changes to the authoritative language entry
- [Module "Translate"](/docs/desktop/ui/modules/translate) — Propose changes to the non-authoritative language entry
- [Module "Review"](/docs/desktop/ui/modules/review) — Review change requests
- [Module "Card"](/docs/desktop/ui/modules/view) — Shows a detailed view of the selected item
- [Module "Graph"](/docs/desktop/ui/modules/map) — Shows registry items as a graph (early preview)

## Panels

- [Panel "Item"](/docs/desktop/ui/panels/item) — General registry item properties
- [Panel "Language"](/docs/desktop/ui/panels/language) — Choose language to view or edit localized versions
- [Panel "Relationships"](/docs/desktop/ui/panels/relationships) — View and manage concept relationships
- [Panel "Revisions"](/docs/desktop/ui/panels/revisions) — Lists revision history
- [Panel "Selected Revision"](/docs/desktop/ui/panels/selected-revision) — Describes the selected revision
- [Panel "My Draft Change Requests"](/docs/desktop/ui/panels/my-draft-change-requests) — Select which CR to add edits to
- [Panel "Concept Summary"](/docs/desktop/ui/panels/concept-summary) — Summary of selected concept revision
- [Sidebar](/docs/desktop/ui/panels/sidebar) — Collapsible sidebar
- [Top Panel](/docs/desktop/ui/panels/top-panel) — Collapsible top panel

## Widgets

- [Registry Item ID](/docs/desktop/ui/widgets/item-id) — Internal identifier (read-only)
- [Registry Item List](/docs/desktop/ui/widgets/registry-item-list) — List with select and activate
- [Representing Designation](/docs/desktop/ui/widgets/representing-designation) — First preferred/admitted designation
- [Main Module View](/docs/desktop/ui/widgets/main-module-view) — Main view of selected module
- [File Reveal Button](/docs/desktop/ui/widgets/file-reveal-button) — Show raw data file in Finder/Explorer
- [Settings Button](/docs/desktop/ui/widgets/settings-button) — Opens settings window
- [Synchronize Button](/docs/desktop/ui/widgets/sync-button) — Syncs with main data repository
- [Create Draft Change Request](/docs/desktop/ui/widgets/new-draft-change-request-button) — Initialize a new CR
