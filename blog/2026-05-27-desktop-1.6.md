Glossarist Desktop 1.6.14 is the latest stable release of the desktop viewer and editor for concept registries.

## What it does

Glossarist Desktop is a native application for managing concept registries. It supports:

- **Browsing** — View concepts in a list or graph view
- **Creating** — Propose new concept entries
- **Editing** — Modify existing concepts in the authoritative language
- **Translating** — Add or edit non-authoritative language versions
- **Reviewing** — Review and accept/reject proposed changes as register manager
- **Synchronization** — Sync with Git-based data repositories
